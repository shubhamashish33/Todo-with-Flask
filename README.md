# Todo Web App

Web Based Todo App made with Flask Module and SQLAlchemy

## Features

- Database Support
- Easy Updatation of Task
- Design for All Screen Size
- Easy Manage your Task

## Installation

- Clone the Repository
- Install Dependencies
- Run Flask Server
